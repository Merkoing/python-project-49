### Brain-games

Brain-games - simple console games in python.

### Install

```bash
$ git clone https://github.com/Merkoing/python-project-49.git
$ cd python-project-49
$ make package-install
```

### Run games


```bash
$ brain-games
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-progression
$ brain-prime
```
